# Saeed Furniture AI State Ledger

This file maintains the continuity of the Saeed Furniture project. It serves as the primary reference for the technical stack, implemented features, and project evolution.

## [ARCHITECTURE]
- **Frontend**: Single Page Application (SPA) using HTML5, Vanilla CSS, and Tailwind CSS (CDN).
- **Logic**: Alpine.js for state management and interactivity.
- **Backend**: Node.js/Express API (located in `/saeed-backend`) handling lead generation, session tracking, and email/WhatsApp notifications.
- **Data**: JSON-driven architecture (`content.json` for CMS, `catalog.json` for product hierarchy).
- **Localization**: Full RTL (Arabic) and LTR (English) support with dynamic language switching.

## [LOCKED FEATURES]
### Authentication & Portals
- **Email/Password Auth**: Pivoted from SMS OTP to email/password. Auth modal has two tabs: "Sign In" and "Create Account". OTP logic removed entirely.
- **Auth Trigger Relocated**: Auth modal no longer fires on category selection. Catalog is fully open. Auth gates only the "Order" / final action button.
- **Supabase Authentication**: Live integration active using project `xbghhxwndctfmpalvdzy`.
- **Client Auth Intercept**: Unauthenticated users clicking Level 1 (Main Categories) trigger a frosted-glass Auth Modal preventing unauthenticated selection progression.
- **Standalone Auth Page**: Dedicated `/auth` route for staff featuring a split-screen layout with a slow-panning Majlis image.
- **RBAC Middleware**: Client-side role validation routing users to specific dashboard views based on their Supabase `raw_user_meta_data.role`.
- **Three Dashboards**: Separate HTML structures (`/dashboard/client`, `/dashboard/admin`, `/dashboard/production`) supporting Order Tracking, Kanban Dispatch, and Manufacturing Task management.
- **Dynamic Client Dashboard**: Two states — Empty (new user) with "Begin Selection" CTA, and Active (returning user) with order cards, a 5-stage status tracker, and WhatsApp Concierge banner.
- **Upgraded Admin Kanban**: Fully dynamic pipeline with Column-based filtering (New, Measurement, Production, Delivery). Features real-time search, VIP/Urgent filters, and detailed lead expansion modals.
- **CRM Extension**: Database schema now supports `home_visit_completed` status and `assigned_to` fields for personnel management.
- **Staff Portal Relocation**: Staff Portal link is relocated to the absolute bottom-left corner (labeled "Atelier Login") of the footer to prevent overlap with the floating action buttons.

### UI/UX & Aesthetics (Quiet Luxury)
- **Premium Palette**: Ivory, Bronze, and Gold (#c9a96e) design system.
- **Hero Sheen Reveal**: `animate-sweep` shimmer effect on the hero section.
- **Arabesque Backgrounds**: Subtle SVG patterns (`arabesque-bg`) applied to section backgrounds.
- **Gold Thread Lines**: Dynamic background accents (`gold-lines-bg` and `gold-strip`) for a premium feel.
- **Pinterest Selection View**: Isolated, high-contrast configurator mode with smooth transitions.
- **RTL Form Underlines**: Progressive disclosure underlines (`rtl-input-line`) that animate on focus.
- **Animated Success Checkmark**: `animate-draw-check` SVG animation for lead submission confirmation.
- **Glassmorphism**: Extensive use of `backdrop-blur` and translucent overlays for depth.
- **Mobile Optimization**: Fully responsive layouts with snap-scroll carousels, `touch-pan-x` swipeability, and `min-h-[44px]` tap targets.
- **Dynamic Footer Routing**: Header contact CTA smooth-scrolls to an elegant data-rich footer and highlights the WhatsApp CTA with a "Gold Zoom Effect".
- **5-3-2 Gallery Architecture**: Dynamic JSON-driven image carousels displaying strictly deduplicated 5 (Types), 3 (Subtypes), and 2 (Variations) images per level.

### Functional Logic
- **Progressive Lead Capture**: Real-time `autoSave` functionality that drafts leads to the backend as the user completes the form.
- **Session Tracking**: Unique session and order ID generation (`SF-XXXX`) for tracking user journeys.
- **Multi-Level Configurator**: Nested selection logic: Type -> Configuration -> Variation.
- **Auto-Forwarding**: Intelligent step skipping for single-option subgroups to reduce friction.
- **WhatsApp Integration**: Floating CTA with click-tracking and pre-filled message support.
- **RTL/LTR Toggle**: Seamless language switching with local storage persistence.
