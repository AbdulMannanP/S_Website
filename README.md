# Saeed Furniture - Luxury Made-To-Order Furniture

This project is a high-end, mobile-optimized lead generation website for Saeed Furniture, targeting the Saudi Arabian market.

## Project Structure

- `frontend/`: The client-side application built with HTML, CSS (Vanilla), and Alpine.js.
- `saeed-backend/`: The server-side application built with Node.js and Express.

## Features

- **Luxury Design**: Refined Ivory/Bronze color palette and minimalist UI.
- **Selection Room**: A glassmorphic, distraction-free environment for product selection.
- **Lead Capture**: Efficient data collection for custom furniture inquiries.
- **Multilingual Support**: Arabic and English RTL support.

## Getting Started

### Backend
1. Navigate to `saeed-backend/`.
2. Run `npm install` to install dependencies.
3. Configure your environment variables in `.env`.
4. Run `npm start` to start the server.

### Frontend
1. Open `frontend/index.html` in your browser or serve it using a local server.
